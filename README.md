# Stash - 개발자를 위한 스니펫 매니저

**Stash**는 개발자의 생산성을 극대화하기 위해 설계된 가볍고 빠른 데스크탑 스니펫 관리 도구입니다. <br/>
자주 사용하는 코드 조각, 명령어, 민감한 정보를 로컬에 안전하게 저장하고 즉시 검색하여 워크플로우를 혁신하세요.

---

## 🚀 핵심 기능

| 기능 | 설명 |
| :--- | :--- |
| **⚡️ 초고속 검색** | 퍼지 매칭(Fuzzy Matching)을 지원하여 입력 즉시 결과를 보여줍니다. (Debounce 최적화) |
| ** snippets 관리** | `CODE` 와 `CMD` 모드를 통해 코드 스니펫과 터미널 명령어를 문법 강조와 함께 깔끔하게 관리합니다. |
| **🔒 시크릿 스태시** | API 키, 비밀번호 등을 **AES-256 암호화**로 안전하게 보관하고, 클릭 한 번으로 안전하게 복사합니다. |
| **🎨 테마 지원** | 개발 환경에 맞춘 다크(Dark) 및 라이트(Light) 모드를 완벽하게 지원합니다. |

## 🛠️ 강력한 개발자 도구

| 유틸리티 | 설명 |
| :--- | :--- |
| **🔍 Diff Checker** | 두 텍스트나 코드를 좌우(Split View)로 비교하고, 단어 단위 차이점을 시각적으로 강조합니다. |
| **📝 Markdown Previewer** | GitHub 스타일 테마가 적용된 실시간 마크다운 에디터와 미리보기 기능을 제공합니다. |
| **📡 HTTP Client** | CORS 제한 없이 백엔드 API를 테스트할 수 있는 경량 HTTP 클라이언트입니다. (GET, POST 등 지원) |
| **🎨 Code to Image** | 코드를 아름다운 그라데이션 이미지로 변환하고 PNG 파일로 즉시 내보낼 수 있습니다. |
| **✍️ 텍스트 & JSON 유틸**| SQL 포맷팅, JSON Pretty/Minify 등 텍스트 처리를 위한 다양한 유틸리티를 제공합니다. |
| **🕒 타임스탬프 변환기** | Unix Epoch 타임스탬프와 날짜(Date)를 실시간으로 양방향 변환합니다. |
| **🎲 데이터 생성기**| UUID v4, 보안 랜덤 문자열 등 개발에 필요한 데이터를 빠르게 생성합니다. |

## ⚙️ 워크플로우 및 생산성

| 기능 | 설명 |
| :--- | :--- |
| **⚡️ 글로벌 퀵 메뉴** | 어느 창에서든 `Ctrl+Shift+Space` 단축키로 Stash를 즉시 실행하여 컨텍스트 전환 비용을 최소화합니다. |
| **💾 데이터 관리** | 전체 스니펫을 JSON 파일로 가져오거나(Import) 내보낼 수(Export) 있어 백업과 마이그레이션이 편리합니다. |
| **🔄 데이터 초기화**| 한 번의 클릭으로 모든 로컬 데이터를 안전하게 초기화합니다. (확인 절차 포함) |
