# Stash - 개발자를 위한 스니펫 매니저

**Stash**는 개발자의 생산성을 극대화하기 위해 설계된 가볍고 빠른 데스크탑 스니펫 관리 도구입니다. <br/>
자주 사용하는 코드 조각, 명령어, 민감한 정보를 로컬에 안전하게 저장하고 즉시 검색하여 워크플로우를 혁신하세요.

---

## 🚀 핵심 기능

| 기능 | 설명 |
| :--- | :--- |
| **⚡️ 초고속 검색** | 퍼지 매칭(Fuzzy Matching)을 지원하여 입력 즉시 결과를 보여줍니다. (Debounce 최적화) |
| **snippets 관리** | `CODE` 와 `CMD` 모드를 통해 코드 스니펫과 터미널 명령어를 문법 강조와 함께 깔끔하게 관리합니다. |
| **🔒 시크릿 스태시** | API 키, 비밀번호 등을 **AES-256 암호화**로 안전하게 보관하고, 클릭 한 번으로 안전하게 복사합니다. |
| **🎨 테마 지원** | 개발 환경에 맞춘 다크(Dark) 및 라이트(Light) 모드를 완벽하게 지원합니다. |

## 🛠️ 강력한 개발자 도구

| 유틸리티 | 설명 |
| :--- | :--- |
| **🔍 Diff Checker** | 두 개의 텍스트나 코드를 나란히 비교(Split View)하여 단어 단위의 차이점을 시각적으로 강조해 줍니다. |
| **📝 마크다운 미리보기** | GitHub 스타일의 테마가 적용된 실시간 마크다운 에디터 및 미리보기를 제공합니다. |
| **📡 HTTP Client** | CORS 제약 없이 백엔드 API를 테스트할 수 있는 경량 HTTP 클라이언트입니다. (GET, POST 등 지원) |
| **🎨 Code to Image** | 작성한 코드를 예쁜 그라데이션 이미지로 변환하여 PNG 파일로 즉시 내보낼 수 있습니다. |
| **✍️ 텍스트 & JSON 도구** | SQL 포맷팅, JSON Pretty/Minify 등 다양한 텍스트 처리 유틸리티를 제공합니다. |
| **🔐 JWT 디버거** | JSON Web Token(JWT)을 오프라인에서 안전하게 디코딩하고 검사합니다. |
| **🔤 Base64 변환기** | UTF-8을 지원하는 안전한 Base64 인코더 및 디코더입니다. |
| **🏗️ JSON to Interface** | JSON 데이터를 기반으로 TypeScript 인터페이스를 자동으로 생성합니다. |
| **🕒 타임스탬프 변환기** | Unix Epoch 타임스탬프와 날짜 간의 변환을 실시간으로 지원합니다. |
| **🎲 데이터 생성기** | UUID v4, 안전한 랜덤 문자열 등 개발에 필요한 데이터를 빠르게 생성합니다. |

## ⚙️ 워크플로우 및 생산성

| 기능 | 설명 |
| :--- | :--- |
| **⚡️ 글로벌 퀵 메뉴** | 어떤 창에서든 `Ctrl+Shift+Space` 단축키로 Stash를 즉시 실행하여 컨텍스트 전환을 최소화하세요. |
| **📜 인피니티 노트** | 자동 저장 및 강력한 검색 기능을 지원하는 영구적인 메모장(Scratchpad)입니다. |
| **💾 데이터 관리** | 모든 스니펫을 JSON 파일로 손쉽게 내보내거나 가져올 수 있어 백업 및 이동이 편리합니다. |
| **🔄 데이터 초기화** | 클릭 한 번으로 로컬 데이터를 안전하게 초기화할 수 있습니다. (확인 절차 포함) |
